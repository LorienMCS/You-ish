About You-ish

You-ish is a capstone project for the Full Stack Web Development program at Galvanize.

I used five APIs:

  • Giphy API
  • Today In History API
  • iTunes API
  • OMDb API
  • Wayback Machine API

A couple of my project's buttons were styled using Beautiful Buttons for Twitter Bootstrappers

My project's cool background came from Subtle Patterns

I designed the site icon for this project using X-Icon Editor

Thanks for visiting!
